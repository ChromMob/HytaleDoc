JavaPlugin

abstract class JavaPlugin extends PluginBase

The main class for Java-based plugins. All Java plugins must extend this class. Java plugins have access to the full Hytale plugin API including:

  • Event registration via EventRegistry
  • Command registration via CommandRegistry
  • Component and system registration via ComponentRegistry
  • Asset management via AssetRegistry
  • Codec registration via CodecRegistry
