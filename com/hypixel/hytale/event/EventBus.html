EventBus

public class EventBus implements IEventBus

The central event dispatcher for the Hytale server. Provides both synchronous and asynchronous event handling with:

  • Priority-based event ordering (HIGH, NORMAL, LOW)
  • Keyed events for per-entity/per-player events
  • Global and unhandled event handlers
  • Performance timing metrics
